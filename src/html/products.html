Hard Coils (Resin Rich up to 18kV)

The "HighMicaHard-C" system is a resin-rich class F/H insulation system for motors and generators up to 18kV. It shows excellent electrical, thermal, and mechanical properties and can resist severe environmental conditions.

Green System (Up to 1 kV)
HighMicaGreen速

The "HighMicaFlex" system is a class F/H insulation system for motors and generators up to 1 kV if no need for any main wall insulation tape.

Soft Coils (VPI up to 15kV)
HighMicaSoft速

The "HighMicaSoft" system is an insulation system for Vacuum Pressure Impregnation in class H (depending on the VPI resin) for motors and generators. The insulation materials from this system are compatible with most epoxy and polyester VPI resins. It is however always advisable to check the compatibility of the HighMicaSoft system with the particular VPI resin. Dependent on the properties of the VPI resin, the Softcoil-C system can be used for motors and generators up to 15 kV. With an appropriate VPI process, the Softcoil-C system results in a high class winding with excellent dielectric and mechanical properties.

Hybrid System (Up to 6kV)
HighMicaFlex速

The "HighMicaFlex" system is a hybrid class F insulation system for motors and generators up to 6 kV if no Tangent Delta and DLA measurements are required. The system offers the main advantages of both the resin-rich coil (electrical, thermal, and mechanical stability after curing without impregnation) and of the soft coil (flexible during winding and continuous insulation). The HighMicaFlex coil combines an economical price with easy winding properties and an excellent quality level. Because the Tangent Delta and DLA values of the Thermoflex coils might be higher than the EN50209 values, the use of the Thermoflex insulation system is advised only up to 6 kV.
